# OpenSpec

This directory contains the baseline specification set for the `private-chef` project inside this repository.

## Current baseline specs

- `specs/frontend/spec.md`
- `specs/backend/spec.md`
- `specs/shared/spec.md`

## Add a new change

Create a new folder under `openspec/changes/<change-name>/` with:

- `proposal.md`
- `tasks.md`
- `design.md` when needed
- delta specs under `specs/`
